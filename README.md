# focus
Projeto para dominar Java e Spring Boot , implementando novas funcionalidades, agora com a mentalidade de um SaaS para empresas. Esse projeto é mais focado em arquitetura limpa e integreção de API de pagamento com Efí Bank, API ViaCep para salvar informações do usuário. SOLID será aplicado na prática nesse projeto, bem como testes unitários.
